void

void is a type that represents nothing. For example, to define a function that returns nothing (aka procedure or subroutine), make it a void type: 
void f() {
  // ...
}

draw

draw() is a function that is repeatedly called during sketch execution. It is equivalent to having the following loop executed: 
setup();
while (noloop() has not been called) {
  draw();
}

setup

setup() is a function that is called once at the beginning of the sketch execution.

See also draw().

textSize

textSize(size) changes the text size to be used by the next text() call. It does not produce any output itself. Arguments:
  • size an integer specifying the text size. Typical values between 5 and 50.

text

text(message, x, y) prints out the text message at position (x, y). Arguments:
  • message — a string to print.
  • x — x coordinate of where to print text.
  • y — y coordinate of where to print text.

str

str(obj) converts any object to string.
